transforming plugin dox

Usage: 
$("#selector").transform(options);
Or call a sprecific method 
$("#selector").transform(methodname [, options]);
General tansforming options
rotateX rotation arount x axis (horizontal, in degrees)
rotateY rotation arount y axis (vertical, in degrees)
rotateZ rotation arount z axis (depth, in degrees)
rotate like rotateZ, excludes rotateX,Y,Z
scaleX scale x axis (horizontal, 0..1)
scaleY scale y axis (vertical, 0..1)
scale scale x and y axis (0..1), excludes scaleX,Y
skewX skew x axis (horizontal, degrees)
skewY skew y axis (vertical, degrees)
skew like skewX, excludes scaleX,Y
Methods
reset reset to defaults transform state defined on first call. If no options were set, the transformations get removed.
animate start animation to specified animation state
Additional options
animateto complete set of general options to animate to. See table above.
easing jquery easing function for animation.
duration duration in ms for animation.
reverse animate to defaults, not animateto.
easingreverse jquery easing function for reverse animation.
durationreverse duration in ms for reverse animation.
step function(t,$element) - optional callback to get called on each animation step.
complete function($element) - optional callback to get called when animation is completed.
beforeanimatein function($element) - optional callback to get called when animation to animateto is started.
afteranimatein function($element) - optional callback to get called when animation to animateto is ended.
beforeanimateout function($element) - optional callback to get called when reverse animation is started.
afteranimateout function($element) - optional callback to get called when reverse animation is ended.
toggle Toggle between default transform state and animateto-state. Same parameters as animate.
hover Toggle between default transform state and animateto-state on mouseover/out. Same parameters as animate.
